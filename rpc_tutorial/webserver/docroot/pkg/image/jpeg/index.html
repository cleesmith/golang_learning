Package jpeg

import "image/jpeg"

Overview ▹

Overview ▾

Package jpeg implements a JPEG image decoder and encoder.

JPEG is defined in ITU-T T.81: http://www.w3.org/Graphics/JPEG/itu-t81.pdf.
